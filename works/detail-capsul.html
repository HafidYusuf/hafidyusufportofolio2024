CONTEXT

[CAPSUL] is a fashion / technology / lifestyle brand, that defines modern urban living in a post pandemic world. The brand consists of the newest style trends, iconic prints, functional design and technology with high end production and materials.

GOALS

Creating a simple, sophisticated and of course the most important part to make a an ecommerce website is making it easy to use yet awesome ecommerce platform to sell a new clothing brand

SO, THE THING IS...

Creating a simplified tunnel to order a product and reducing bounce rate is what i aiming for, gladly we use shopify wich the checkout tunnel is already been built by them, so i have nothing to worry about the product checkout tunnel

I want to make the user says "WOW" when they see the website for the first time

What i always do when creating a website is, creating a catchy, and standout header section, because... first impression matter, so what i do is creating an interactive image pop ups of the products, while user move their cursor and starting the website from blank and the content is shows up one by one, so it has a feeling that the website is being built while the user is opeing it

HERE IS SOMETHING COOL

I also creating an interactive worldmap that lights up when user hover it and also show a coordinate of the city in every country. Adding a collection page to create an interactive gallery of each product is a nice thing to have in an ecommerce website
